# 2 객체의 종류
VO, DTO, DAO, 엔티티란?
## 저연차 개발자의 답변
1. VO :읽기 전용 객체
2. DTO : 걔층 간 데이터 교환 (DB에 데이터를 넣거나 불러올 때)
3. DAO : DB 접근 객체
4. 엔티티 : JPA의 @Entity, 테이블과 1대1 대응 ~> JPA가 없으면 설명할 수 없는 개념인건가?
